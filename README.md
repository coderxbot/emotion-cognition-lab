# Emotion-Cognition Laboratory

## 项目概述

这是一个致力于情感认知科学研究的学术实验室项目，专注于探索**人类情感的神经生物学机制**、**情感调节策略**和**认知-情感交互**。

## 研究方向

### 核心研究领域
- **情感识别与分类** - 面部表情、语音、生理信号识别
- **情感神经生物学** - 脑成像数据分析，神经关联研究
- **情感调节机制** - 认知重评、表达压制等策略的效果评估
- **跨文化情感研究** - 不同文化背景下的情感表达差异
- **人机交互中的情感** - AI系统中的情感识别与响应

## 数据集

本项目使用的主要数据集：
- **AffectNet** - 大规模面部表情数据库
- **RAVDESS** - 多模态情感语音数据库
- **IAPS** - 国际情感图片库
- **自采集数据** - 研究特定实验的数据
