# Contribution Guidelines

Thank you for considering contributing to the Emotion Cognition Lab project! To help guide you through the contribution process, we have outlined the following guidelines:

## Reporting Issues
- Check the existing [issues](https://github.com/coderxbot/emotion-cognition-lab/issues) to see if your issue has already been reported.
- If not, please create a new issue with a descriptive title and detailed information about the problem. Include steps to reproduce the issue, expected results, and any relevant screenshots.

## Submitting Code
1. **Fork the repository** by clicking the "Fork" button at the top right of the GitHub page.
2. **Clone your fork** to your local machine:  
   `git clone https://github.com/YOUR-USERNAME/emotion-cognition-lab.git`
3. **Create a new branch** for your feature or fix:  
   `git checkout -b feature/your-feature-name`
4. Make your changes and **commit** them:  
   `git commit -m "Add a descriptive commit message"`
5. **Push your changes** to your fork:  
   `git push origin feature/your-feature-name`
6. Go to the original repository and submit a **pull request** from your branch.

## Improving Documentation
- Documentation improvements are always welcome! To contribute to the documentation:
  - Identify areas that need clarification or enhancement.
  - Follow the above steps to fork the repository and make a pull request with your changes.

## Sharing Research Results
- If you wish to share research results that involve this project, please submit a detailed report including:
  - The context of your research.
  - The methodology used.
  - A summary of the results.
- This report can be submitted via an issue or as a pull request with relevant documentation.

## Ethical Guidelines
- Contributions should adhere to the highest standards of ethical behavior. Please:
  - Respect the privacy and confidentiality of all individuals involved in research.
  - Avoid any form of plagiarism and ensure proper citations.
  - Follow applicable laws and institutional guidelines when conducting research.

Thank you for your contributions and commitment to making the Emotion Cognition Lab project better!
